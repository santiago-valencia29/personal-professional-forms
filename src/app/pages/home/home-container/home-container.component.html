Inicio

Bienvenido al panel de formularios. Aquí podrás explorar un sistema diseñado para registrar y visualizar datos personales y profesionales, utilizando tecnologías de última generación.

Este proyecto está construido utilizando Angular 17.3.12, Bootstrap 5.3.2-Responsive Design, Angular Material, ReactiveForms, SweetAlert 2 y NGXS para el manejo del estado.
